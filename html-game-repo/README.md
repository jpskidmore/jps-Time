# HTML Game Repo - Sky Dodge

A lightweight browser game built with plain HTML, CSS, and JavaScript.

## Run locally

Open `index.html` directly in your browser, or use any static file server.

## How to play

- Move left: `←` or `A`
- Move right: `→` or `D`
- Avoid falling rocks as long as possible.
- Your score increases each time a rock passes you.

High score is saved in browser local storage.
