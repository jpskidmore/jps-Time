# HTML Game Repo - Sky Dodge

A lightweight browser game built with plain HTML, CSS, and JavaScript.

## Open it from your iPhone/iPad Safari with one link (GitHub Pages)

GitHub does **not** run `index.html` directly from the code view page. To open it as a real website, publish with GitHub Pages once, then use the generated link forever.

1. Push this repo to GitHub (branch: `main`).
2. In GitHub, go to **Settings → Pages**.
3. Under **Build and deployment**, set **Source** to **GitHub Actions**.
4. Go to the **Actions** tab and run/confirm **Deploy Sky Dodge to GitHub Pages**.
5. Open this URL on your device (replace `YOUR-USERNAME`):
   - `https://YOUR-USERNAME.github.io/jps-Time/`

After that, you can just tap that URL in Safari and it will launch the game.

## Run locally

Open `index.html` directly in your browser, or use any static file server.

## How to play

- Desktop move left: `←` or `A`
- Desktop move right: `→` or `D`
- Mobile Safari: tap **Left** and **Right** buttons under the game
- Avoid falling rocks as long as possible.
- Your score increases each time a rock passes you.

High score is saved in browser local storage.
